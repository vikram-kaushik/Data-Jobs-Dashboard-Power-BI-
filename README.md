# Data Jobs Dashboard (Power BI)

An interactive Power BI dashboard analyzing 2024 data job postings — covering job counts, salary trends, and role comparisons across Data Analyst, Data Scientist, Data Engineer, ML Engineer, and related titles.

## 📊 Dashboard Overview

**Key metrics tracked:**
- **Job Count:** 479K postings analyzed
- **Job Rating:** 3/5 average
- **Median Yearly Salary:** $113K
- **Median Hourly Salary:** $48

**Visuals included:**
- Monthly job posting trend for 2024 with a linear trendline
- Hourly vs. yearly median salary scatter plot across job titles
- Highest paying data roles (bar chart)
- Job title breakdown table with job count, salary, and mini trendlines

## 🔍 Drill-Through: Job Title Details

<img width="1450" height="809" alt="Screenshot 2026-07-06 124046" src="https://github.com/user-attachments/assets/d298e669-f60b-4abe-8e02-5c88e2b2fe57" />

Clicking into any job title (e.g., Machine Learning Engineer) opens a detail page showing:
- Median yearly & hourly salary gauges
- WFH availability %
- Degree requirement mentions %
- Health insurance offering %
- Global job posting locations (map)
- Job platform source breakdown
- Job schedule type (full-time/part-time/contract)

## 🛠️ Tools Used
- **Power BI** – data modeling, DAX measures, interactive visuals, drill-through pages
- **Power Query** – data cleaning and transformation

## 📁 Dataset
Job postings dataset (2024) sourced from Luke Barousse's Power BI for Data Analytics course.

## 💡 Key Insights
- Job postings peaked mid-year and declined sharply toward Q4 2024.
- Machine Learning Engineer and Senior Data Engineer roles command the highest yearly salaries (~$155K–$156K).
- Most Machine Learning Engineer roles are full-time (94%), rarely mention degree requirements (4%), and offer limited WFH flexibility (18%).
